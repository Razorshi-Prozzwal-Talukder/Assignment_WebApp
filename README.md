# How to Run this Application
---------------------------------

```bash
-- Download that [Project](https://github.com/Razorshi-Prozzwal-Talukder/Assignment_WebApp).
-- Open that solution (WebAppAssignment.sln) by [Visual_Studio](https://visualstudio.microsoft.com/vs/).
-- Make the Database and tables in [Microsoft_SQL_Server_Management_Studio](https://www.microsoft.com/en-us/sql-server/sql-server-downloads).
-- Try to keep the DB name (Assignment) and Table name (Guest_tbl, User_tbl) 
   and their attributes as same as sln. Otherwise, you would change the name in code section.
-- Remove Model1.edmx from sln and Add your model from your Database.
-- Build the Solution and Run.
```
